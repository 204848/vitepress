# 异界的碎片(01)

白辰:"呃，那个，九颜，似乎你们这个界域附近好像还有个小碎片欸，你知道吗？"（九颜）：“？什么碎片啊，我好像没听过吧，说说？”，“就是这个啊”，他打开了星图把那个界域碎片指给她和秦岚看，“emm，这个不会是以前人妖的冲突战争给打碎出来的吧？”九颜说，同时揉了揉白辰的耳朵，“呃.....”白辰脸红了红，“我不知道啊，这里毕竟我不了解嘛”，秦岚：“人和妖还有这么大规模的冲突？现在根本做不到吧”，“对于这个我了解也不多，只知道一点，在几千年前你们人族其实是可以修仙的，有灵气有法则，但是你们这毕竟是下界存在境界上限，但是你们人族一些宗门强行突破透支界域所有资源最后虽然到了高的境界但是也透支了界域所有底蕴，修仙完全绝迹了，而那些宗门则直接离去了去了界域边缘处...”，"哦..."（白辰），“你小子，可爱呢”（九颜），
