## 基础
名字:白辰

性别:男

年龄:22（从出生算起未转换）

种族:半妖(狐妖+人族)

## 定位
这是一个真元老级的人物，贯穿了整个幻想设定在定位上也是属于主角吧(可能吧因为没有涉及过)，同时他可能对我的意义不一定是一个普通的人物设定了，可能是我内心的一些小想法


## 背景
1.首先在[前面](../世界观.md)说过这个世界有不同的设定，而在目前的设定中白辰前期是属于一个科技世界的一个科学家(做出过很多贡献如虫洞穿越平行宇宙，核聚变等等)，然后后面脱离了这个世界(是完全脱离了基本没有任何联系，且其实这个身份更多是挂名因为没有提过多的科技成分)，去各个宇宙游荡去了(也可能是有时受伤了)，去看看不同的世界
::: tip 提示
但他这个不是什么旅客旅人只是去看看不同风貌，当然这个原因也是不用在意的，因为没怎么设定不过大概的也有
:::

至于脱离的原因一个是因为那个科技世界想强迫他研究武器但他本身的性格又不想而且他也听说其实他不是这个世界的而且在外也有个亲人(2里讲了)但是他也不知道在哪而且在这个科技世界虽然他的贡献比较大但是也还是压迫他于是就脱离了

然后相对前期的背景就是这样

2.在近期下然后他还有一个姐姐对他也比较好然后也是最近才找到他的至于为什么要找嘛

(1):首先因为在白辰他出生后他的父母就把他丢给了他的姐姐(原因不用管为什么是因为容易阴谋论而且一写就是长篇大论)，而他的姐姐怕照顾不好他(因为他姐姐的血脉是纯的狐妖血脉没有人的怕照顾方法不一样而且怕她没那么多时间好好照顾他)于是就把他给给寄托在了一个科技世界的朋友那(就是上面的)然后就暂时因为一些事离开了(就是因为招惹了一些别的人/妖这个不是阴谋论只是一些正常的东西了只是正常的仇恨，而且她自己也要修炼)

(2):再去找他是因为想了想这么久了他应该也长大了一点(因为她也不知道他的血脉对白辰的影响)于是又去寻他了但那时他已经离开了那个科技世界所以她也是找了好一阵才找到的


## 血脉相关
1.首先他是一直有妖的相关血脉的但是在以前的时候一直没有表现出来只是在那里藏着就连他自己也不知道以及了解自己的这个特点在以前一直以为自己就是普通的人但是后面的时候妖的血脉慢慢表现出来了才知道自己不是完全是人(当然不要提什么基因检测等等科技)

2.妖的血脉对他的影响基本上是只对他的外貌有影响，其他在如实力/体质等等没有受到影响是普通人的实力并没有增强，当然他的感知力稍微好了点对于妖的气息以及修为境界更加敏感，但是虽然他的感官更加的敏锐(特别是对妖力的感知)但是因为反应力没有增幅所以很多时候就算感觉到了但是也没有太大用

3.其实他的血脉本身是很强的修炼天赋也应该很强算是狐妖里最顶级的血脉了，但是因为人的血脉和妖的血脉有点冲突导致的无法修炼，同时对"灵气"也不太兼容

4.同时妖的血脉没有影响到他的思维他的思维<span class="marker-evy">也是人的思维</span>没有受到影响
::: tip 提示
实际上在我这里因为每个世界都有点小不一样所以大概不是每个都有"妖"的性格之类的
:::

5.关于血脉的来源也不用纠结 ~~(因为我也没有设定(￣y▽,￣)╭ )~~ 额，因为我不想设计这些纠缠，很容易就有阴谋论在里面以及有大义凛然之类的精神所以我不想解释，最好的方式我认为就是<span class="marker-evy">避免</span>

6.当然血脉也带来了一些小东西比如对妖力/妖气的感知比较敏感，对修为也比较的敏感可以感知到一个人/妖的修为之类的，也可以感知到妖气之类的躁动

## 性格
::: tip 提示
虽然他的年龄比较小但是思想也比较成熟了不是小孩思维了虽然后面的性格可能看起来有点不成熟但是其实是他本来的思维而不是不成熟
:::

1.在性格这一方面他是比较<span class="marker-evy">温柔善良老实的</span>，他一直是比较善良的对于很多事都是比较善良的比如对他人总有点善意的，同时他也是不容易生气的(我的印象里好像没有生气过)

2.然后他也是比较容易害羞的 ~~(容易红温?但是也不算红温吧)~~ 对于有时容易害羞，因为..怎么说呢，反正他对性有关的知识基本完全没有了解 ~~(再叠加上害羞的性格怎么怪怪的)~~

3.然后他也是比较天真的(虽然看起来可能想思想不成熟但实际上就是他本性)，比较天真

4.然后他也是因为比较善良爱好和平所以对于一些血腥场面之类的也是比较有点反感一点的对于一些场面也是没怎么见过的

5.他也因为前面的带点善意所以对于一些事想的也没那么深，所以也容易被坑以及"拐骗"

6.温柔不等同于爱哭之类的这是两个概念，温柔和善良不是爱哭

## 认知设定

1.其实他对科技是比较的了解的比如说核聚变之类的，**但是他的思维没有被科学所取代！！**他**不会用科学来解释所以食物，科学只是一个小工具不是一切与主要**他的思维还是正常的和[上面](#性格)的善意与温柔的，就是正常的思维

2.同时其实他对妖和修仙也是有一定了解的，但是不多,基本没多少知道的主要是关于通用境界划分以及可以看出来借助[上面](#血脉相关)的血脉感知可以感知到大概的修为，以及分出妖和人，以及灵气的存在，但也仅此而已了

## 特点设定(部分)

1.借助血脉他可以感受与分辨出不同的妖和人修为以及不同的妖，同时暂时是不能修炼的

2.因为暂时不太兼容灵气/妖气而且血脉等级比较高所以对于普通的尝试用血脉来压一般是没用的，而且如果是专门针对攻击灵气/妖气的方法与限制也是没用的

3.从目前的设定来看他体内狐妖血脉的占比还在慢慢的提升慢慢的上升，这也会导致他的外貌可能后面还会偏向一点妖
