## 世界背景设定
::: warning 警告 
这里的<sapn class="marker-evy">“世界”</sapn>要看是指的是大的世界就是指母宇宙还是是平行宇宙与子宇宙，这里要区分
:::

首先这整个世界都基于一个物理理论[无限暴涨理论/泡泡宇宙假说](https://baike.baidu.com/item/%E6%B3%A1%E6%B3%A1%E5%AE%87%E5%AE%99%E5%81%87%E8%AF%B4/12649327)，首先具体的原理参考链接或[bete物理版](/小幻想/新想法/新世界观(物理版).md)，简单来说借助了其中对不同宇宙物理常数允许不同而为基础

1.首先在该理论中可以存在大量的小宇宙类似平行世界，每个平行世界各不相同，类似多元宇宙，存在一个无限暴涨的母宇宙而在其中有很多子宇宙，子宇宙就像在一个个小岛在母宇宙的大海上

2.既然在前面说了**可以有多个子宇宙且可以有不同物理常数**那么就可以有不同的宇宙以及规则，如玄幻宇宙，科技宇宙等等，且生命形态也可以不同支持不同种族

3.从此理论上可以得知不同宇宙可以互相融合，此时可以将子宇宙看作小说中的“界域”，而不同的子宇宙间融合便会有不同的接触且因此可能改变双方的宇宙常数以符合如界域融合导致的灵气复苏(伏笔+1)，类似两个泡泡融合在了一起同时因为融合需要时间所以不是瞬间的提供空间

4.因为理论的基础子宇宙的数量可以非常庞大所以可以涵盖大多数的场景因为数量足够再小的事件发生概率也是被允许的

5.一个子宇宙中可能有几个文明但是距离一般较远所以无太多交际大多可以忽略，但是需要时理论上也可以

6.因为有不同子宇宙所以每个宇宙时间流速是可以不一样的，有的相对更快有的相对更慢

## 适应性调整
为了幻想更加合理顺畅所以加了一部分的小设定如下
1.虫洞回归允许通过虫洞来快速跨越空间，也可以去往不同的子宇宙(主要给主角用的)

2.量子纠缠允许传递信息以达到跨子宇宙信息传递(总不能传个信息等到天荒地老吧(* ￣3)(ε￣ *) )

3.曲率引擎允许被使用用来短距离移动用

4.母宇宙中的光速普遍高于其它小宇宙

5.为了解决玄幻世界修炼时间问题利用不同宇宙时间差和广义相对论来统一一下以达到外界时间加速但是主角不会过去太多时间以防止主角寿命限制了，同时不影响别的子宇宙的时间流速了

6.母宇宙内还可以通过特殊方法手动构建一小片区域类似子宇宙

## 小宇宙设定

1.首先每个小宇宙“按理”来说是独立的互相间没有什么特殊联系除非碰撞融合，所以每个小宇宙可以有不同的小世界观以及小的剧情互相独立当然也可以互相干预

2.小宇宙可以有不同的物理常数来有不同的性质，以及可能有不同结局如无生命，不同宇宙死亡结局

3.不同小宇宙间间隔普遍较远但是也是可以互相到达的可以进入别的小宇宙中，当然可能部分小宇宙不支持生存

4.子宇宙处于母宇宙内不同子宇宙可以相互融合当然会导致双方的法则都有点变化

## 世界类型
无固定的世界类型，世界的类型取决于要写的小宇宙的设定，不过可以有一些模板

::: details 修仙灵气
可以引用灵气来写，至于灵气可以看作加了一个新的场方程使用灵气就是和场方程互相作用的结果，有修仙的世界可以是场方程活跃而无法修仙/不好修仙则是场方程不那么活跃
:::

::: details 界域碰撞
类似前面说的不同小宇宙可以互相碰撞融合，且融合不是一瞬间而是慢慢的且可以通过观测来察觉，然后如果是界域碰撞后灵气复苏则可能是两个世界互相作用的结果
:::

::: details 普通小说/游戏
可以直接导入就行了不用多复杂的处理因为比较合理了
:::

::: tip 通用修仙境界划分(暂行)
凡：练气<聚气<筑基<结丹<金丹<凝婴<元婴<化神<大乘
仙：真仙<人仙<地仙<天仙<玄仙<仙尊<仙王<准仙帝<仙帝

**注意这只是一个通用标准所以不是每个小宇宙都是这一套只是可以相互转换了**
:::

## 冲突原因？(去中心化)
[前面说过](#小特点)这个世界观的设定是去中心所以每个子宇宙的事件是不同的而不是固定的，所以冲突也会不同这个无法确定。

去中心化可以增强世界观的创新性也可以扩展更多故事了所以同时没有所谓的最主要的中心小世界，以及什么超级跨宇宙势力

当然人物还是要中心化一点的否则就会不连贯了，主要人物还是有的

## 兼容性
因为这个世界观允许不同性质的子宇宙所以事实上是可以将别的小说的世界观作为子宇宙的小世界观来加入的，其中的人物也可以加入其中其中的剧情由此也可以导入，因此可以实现在一套大世界观下有很多小的别的小说的剧情接入且兼容玄幻与科幻以及日常文，兼容性较强

::: details 抽象
看到一本有趣小说烂尾了？看到小说意难平？直接接入自己的幻想中的一个子宇宙，改剧情/续写以及加入自己的元素，满足想象与喜欢！抚平你的意难平(雾)
:::
你可以将不同小说编入其中，甚至两本小说的世界对抗也是可以的兼容性应该还是没太大问题的的
